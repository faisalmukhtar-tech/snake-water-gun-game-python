# 🐍 Snake Water Gun Game (Python)

A simple command-line game built using Python where the user plays against the computer.  
This project is based on the classic "Snake Water Gun" logic, similar to Rock-Paper-Scissors.

---

## 🎮 Game Rules

- Snake (1) drinks Water → Snake wins
- Water (-1) gets beaten by Gun → Gun wins
- Gun (0) kills Snake → Gun wins

---

## ⚙️ Features

- User vs Computer gameplay
- Random computer choice using Python `random` module
- Simple input system (s / w / g)
- Win, lose, and draw logic
- Beginner-friendly Python project

---

## 🧠 Concepts Used

- Python input/output
- Conditional statements (`if-elif-else`)
- Dictionaries
- Random module
- Basic game logic design
